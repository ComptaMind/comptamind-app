Déploiement Netlify

Pour déployer l'app React sur Netlify:

1. Connectez votre repo GitHub à Netlify
2. Build command: npm run build
3. Publish directory: dist
4. Ajoutez un domaine personnalisé: app.comptamind.fr
5. Dans Ionos, ajoutez un enregistrement CNAME: app → votre-app.netlify.app
6. Remplacez APP_URL par https://app.comptamind.fr dans tous les snippets ci-dessus
